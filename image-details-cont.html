Spectrum of a Protostar from Ophiuchus Cloud

This image shows the low resolution spectra of a protostar, named OPH-31 or YLW 16B, from the star forming cloud Ophiuchus. The shown yellow dotted line is a continuum, that has been used as a baseline to perform the fitting procedure. Using some points as anchor points from absorption free regions of the spectra, we have plotted a 3rd order polynomial for the continuum, which fits perfectly for the analysis of almost all the protostellar spectra.
